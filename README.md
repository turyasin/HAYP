# Kayalar Has Yönetim Platformu (HAYP)

HAYP, endüstriyel mutfak ekipmanları satan ve üretim yapan firmalar için geliştirilmiş kapsamlı bir yönetim platformudur.

## Özellikler

- **Firma Yönetimi:** Müşteri, tedarikçi ve gönderen firma yönetimi.
- **Ürün Yönetimi:** Farklı formatlardaki tedarikçi listelerini (Excel, PDF, Resim) tek merkezde toplama.
- **Tekliflendirme:** Hızlı teklif oluşturma, maliyet/kar analizi ve PDF çıktısı.
- **Sipariş ve Fatura:** Sipariş takibi ve faturalandırma entegrasyonu.

## Teknoloji Yığını

- **Frontend:** Next.js, Tailwind CSS, Shadcn UI
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL (Supabase)
- **Otomasyon:** n8n

## Kurulum

1. Depoyu klonlayın.
2. `backend` klasöründe `.env` dosyasını oluşturun (bkz. `.env.example`).
3. `backend` klasöründe `npm install` ve `npm start` komutlarını çalıştırın.
4. `frontend` klasöründe `npm install` ve `npm run dev` komutlarını çalıştırın.
